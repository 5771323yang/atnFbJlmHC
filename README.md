# Vue+Node.js酒店管理系统 java669

## 项目概述

本项目是一个基于Vue和Node.js开发的酒店管理系统。该系统采用前后端分离的架构，前端使用Vue框架搭配Element-UI组件库，后端使用Node.js搭配Express框架，数据库采用MySQL。

## 技术栈

### 前端

- Vue
- Element-UI

### 后端

- Node.js
- Express

### 数据库

- MySQL

### 开发工具

- Vscode

## 功能模块

系统包含以下主要功能模块：

1. 登录模块
2. 首页
3. 人员管理
4. 房间管理
5. 入住管理
6. 入住信息管理

## 系统角色

系统主要角色包括：

- 管理员：负责系统管理、人员管理、房间管理等工作。
- 前台：负责客人入住、退房等操作。
